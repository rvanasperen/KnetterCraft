Book Art Fixer

Generates an enchanted_book.json by parsing and merging Lullaby's Enchanted Books and Beautiful Enchanted Books resource pack JSONs.

Usage:

1. Copy `enchanted_book.json` from Lullaby into `lullaby.json`, likewise for `beautiful.json`, and place them in this directory
2. Run the script: `python3 main.py`
3. Copy the resulting `output.json` to `/resourcepacks/KnetterCraft Enchanted Books/assets/minecraft/items/enchanted_book.json`
